# Frontend Developer Portfolio

A responsive web portfolio built with HTML, CSS, and JavaScript. Designed with a mobile-first approach using brand colors blue, gray, and black.

## Features

- Responsive design (mobile-first)
- Smooth scrolling navigation
- Mobile hamburger menu
- Contact form
- Skills and projects sections

## Structure

- `index.html` - Main HTML file
- `style.css` - Stylesheet with CSS variables for brand colors
- `main.js` - JavaScript for interactivity
- `assets/` - Directory for images and other assets

## Getting Started

1. Clone or download the project
2. Open `index.html` in your browser
3. Customize the content as needed

## Customization

- Update the hero section with your name and description
- Replace placeholder text in about, skills, and projects sections
- Add your project links and images
- Modify colors in the CSS variables if needed
